# Room202 프로젝트

## 1. 프로젝트 소개
Room202 프로젝트는 같은 고등학교 기숙사(202호) 룸메이트들의 추억을 공유하는 어플리케이션을 만드는 프로젝트이다.
기능은 간단하지만 프론트엔드, 백엔드 분야의 기본기를 다지고 포트폴리오로 사용할 수 있게 완성도 있는 프로젝트를 
만드는 것이 목적이다.
